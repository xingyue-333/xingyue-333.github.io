# 陶平柳 个人主页（模板）

这是一个简单的静态个人主页模板，放到仓库根目录（username.github.io）即可被 GitHub Pages 托管。

快速部署步骤：
1. 将 `index.html`、`styles.css` 放到仓库根目录。
2. （可选）把头像放到 `assets/avatar.png` 并在 `index.html` 中替换路径。
3. 提交并 push 到 `main`（或默认分支）。几分钟后访问 `https://xingyue-333.github.io` 即可看到站点。
4. 如果想使用自定义域名，添加 `CNAME` 文件并写入域名。

如果你希望我直接把这些文件提交到仓库，我可以为你创建并推送一次提交（需要你确认）。
